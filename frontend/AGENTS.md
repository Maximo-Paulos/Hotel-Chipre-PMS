# frontend/AGENTS

- Keep pages and components small and typed.
- Keep API access in `frontend/src/api/` and state in hooks or `frontend/src/state/`.
- Preserve hotel/user scoping in session-aware requests.
- Keep user-facing errors clear and safe.
- Update build, lint, and e2e coverage when UI behavior changes.
